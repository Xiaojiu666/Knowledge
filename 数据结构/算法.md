###1.A+B问题
	Q:给出两个整数a和b, 求他们的和, 但不能使用 + 等数学运算符。
--
	A:不能使用数学运算符，第一时间想到使用其他的运算符，例如 位运算符
	位运算符复习
		与:		a&b  ab都为1 结果为1
		或:		a|b	 只要有一个为1 就为1
		非:		~	 操作数相反
		异或：	a^b	 相同得0 不同得1
	举例：
			3 = 0011		      	
		 +	4 =	0100		^					&
		 -----------------------------------------------------
			7 = 0111 			7 = 0111			0 = 0000
 		//当两个数的与为0时,说明他或的值  必为他俩的和
		//则如果前提为a&b = 0的时候 ， a+b=a^b
		
	举例：
			3 = 0011		d      				c
		 +	3 =	0011		^					&
		 -----------------------------------------------------
			6 = 0110 			0 = 0000			3 = 0011
		//



----

	class Solution 
		{
		public:
		    int aplusb(int a, int b) 
		    {
		        int c=a&b;
		        int d=a^b;
		        return c==0?d:aplusb(c<<1,d);
		    }
		};
